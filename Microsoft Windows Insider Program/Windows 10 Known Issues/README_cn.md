# Windows 10 1903 版本更新已知问题与处理进度汇总

[**English Documentation (英文版本)**](https://github.com/Lingggao/Microsoft-Insider-Program/tree/master/Microsoft%20Windows%20Insider%20Program/Windows%2010%20Known%20Issues)

## 一、概述

Windows 10 1903 版本于 **2019 年 5 月 21 日**正式发布。每当 Windows 发布一年两度的功能更新时，此时产品用户们的已知信息是最少的，因此他们可能会发表严重的负面舆论。我在 1903 版本正式发布前的一周已经预料到可能会出现这一状况，**身为一名光荣的 Microsoft Windows 预览体验成员，我决定挺身而出。**

综上所述，为了防止广大的中国 Windows 用户朋友们因为无法获知有关 Windows 10 Bug 最新处理进度的最新信息而对 Microsoft 产品的质量留下较差的印象，我于 **2019 年 5 月 14 日 (1903 版本正式发布前 7 天)** 在 Microsoft 社区创建了 ***“Windows 10 2019 年 5 月更新 (1903_18362) 已知问题与处理进度汇总”*** 讨论贴：

> 点击 [**此处**](https://answers.microsoft.com/zh-hans/insider/forum/all/bug-%E6%B1%87%E6%80%BBwindows-10-2019-%E5%B9%B4-5/252d0d6a-022c-4bf7-9976-55b57590aee2) 以前往 Microsoft 社区查看此讨论贴。

## 三、信息来源

1. 在 **2019 年 5 月 14 日 - 5 月 21 日**之间，由于此时 Windows 10 1903 版本还没有正式发布，因此汇总帖中的问题主要来源于我个人以及其他 Windows 预览体验成员们。**我将反馈中心应用中我和其他预览体验成员们所发现并报告的问题进行了整理，并通过 Microsoft 社区面向广大中国 Windows 用户朋友们进行了展示。**
2. 在 **2019 年 5 月 22 日**之后，此时 Microsoft 已经正式发布了 Windows 10 1903 版本，并且开设了 [***“Windows 发布运行状况仪表板 ”***](https://docs.microsoft.com/zh-cn/windows/release-information/status-windows-10-1903)，专门用于向用户们提供有关 *“Known issues”* 的相关信息。**我在通过反馈中心整理问题的基础上，同时将 Microsoft 官方在 *“Windows 发布运行状况仪表板 ”* 中提供的英文已知问题翻译为了简体中文，并在第一时间通过汇总帖面向中国 Windows 用户朋友们进行了展示**。

## 四、特点

1. **本地化**：完全使用简体中文进行整理，不再需要中国 Windows 用户朋友们自行进行翻译，极大的提高了浏览效率与整体效果。
2. **及时**：在 Microsoft 通过 *“Windows 发布运行状况仪表板 ”* 发布英文已知问题后，我可以确保在 **24** 小时内将其翻译为中文并整理至汇总帖。
3. **准确**：与 [**“网易有道人工翻译”**](https://f.youdao.com/) 进行合作，确保中文翻译的准确性。同时，对于来自反馈中心或用户朋友们所补充的问题线索，我会使用 **3** 台运行 Windows 10 1903 版本的设备进行测试，确保全部可以成功复现后才会整理至汇总帖。
4. **美观**：充分、恰当的运用了字体格式、颜色等特殊效果。已经得到修复的问题会被标记为**蓝色**，Microsoft 正在调查的问题会被标记为**紫色**，Microsoft 正在处理的问题会被标记为**黄色**。

## 五、最新情况

由于 Windows 10 1909 版本即将正式发布，继续更新 “适用于 1903 版本 Windows 10 系统的已知问题” 已经意义不大。因此，*“Windows 10 2019 年 5 月更新 (1903_18362) 已知问题与处理进度汇总 ”* 讨论贴已于 **2019 年 10 月 18 日**停止更新。

在汇总帖正常运行的数个月时间中，我共计整理了 **35** 个 Windows 10 1903 版本已知问题。**截止至 2020 年 3 月，*“Windows 10 2019 年 5 月更新 (1903_18362) 已知问题与处理进度汇总 ”* 讨论贴共计获得了 264 个赞成票数，总浏览次数为 7,434 次。本讨论贴对 “改善中国地区 Windows 10 用户对 Microsoft 的看法” 的根本需求确实起到了一定的帮助**。
